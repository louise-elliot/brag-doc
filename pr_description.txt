⏺ Summary

  Replace the single-shot save-time reframe 
  with an opt-in, multi-turn AI coach
  conversation. Saving a journal entry no   
  longer triggers any AI call — instead,  
  each entry shows a Talk it through with
  the coach button. Clicking it opens an
  inline coaching panel where the coach
  names self-diminishing patterns (hedging,
  luck-attribution, team-deflection, missing
   scope/audience/outcome) and probes for
  missing detail. The user controls when to
  wrap up by clicking Reframe it now, which
  produces the final reframed entry plus a
  consolidated set of coach-note tags stored
   alongside.
